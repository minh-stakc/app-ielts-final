# IELTS Learning App

A full-stack web app for IELTS exam preparation with adaptive practice exercises and user progress tracking. Containerized with Docker and served via Nginx.

## Features

- Practice exercises for IELTS Reading, Writing, Listening, and Speaking sections
- User authentication and progress tracking
- Dockerized deployment with Nginx reverse proxy

## Stack

- **Frontend**: React
- **Backend**: Node.js
- **Infrastructure**: Docker, Nginx

## Setup

```bash
yarn install
yarn start
```

Or with Docker:
```bash
bash build_and_push.sh
```
